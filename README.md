# ♻️ BloomCycle AI

**AI-Powered Floral Waste Intelligence Platform**

BloomCycle AI is a smart sustainability-focused web application that helps track, analyze, and manage floral waste efficiently. It transforms waste flower data into actionable insights such as revenue estimation, recycling score, and sustainable output recommendations.

---

## 🌟 Features

* ♻️ **Waste Entry Management**

  * Add flower waste details
  * Location-based entry tracking
  * Quantity input in kilograms

* 📊 **Live Dashboard**

  * Total entries
  * Total waste collected
  * Estimated revenue
  * Recycling score

* 📈 **Dynamic Analytics**

  * Live updating progress bars
  * Revenue and waste visualization
  * Auto updates when entries change

* 🔍 **Search System**

  * Search entries by flower type
  * Instant filtering

* 🗑 **Delete Entries**

  * Remove outdated or incorrect entries

* 🤖 **AI Assistant**

  * Smart response system for waste reuse suggestions
  * Flower-based output recommendations

* 🌙 **Light / Dark Mode**

  * Interactive glass toggle switch
  * Smooth theme transition

* ✨ **Modern UI**

  * Liquid glass / glassmorphism design
  * Responsive layout
  * Smooth scrolling tabs

---

## 💡 Problem Solved

Flower waste from temples, weddings, festivals, and local markets is often discarded improperly, causing:

* waste accumulation
* garbage issues
* mosquito breeding
* bad odor
* environmental damage

BloomCycle AI helps convert this waste into:

* incense sticks
* compost
* perfumes
* soaps
* essential oils
